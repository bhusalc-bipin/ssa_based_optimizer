# CS 6810 -- SSA-based Optimizer

**Due Date:** *Friday, February 27, 2026 @ 5pm*

## Project Summary

Your task is to write a program that takes as input an `iloc` program representation generated by the front end of a compiler and performs a set of optimization as given below.

1. convert the program to pruned-SSA form
2. perform dominator-based redundancy elimination
3. perform dead code elimination
4. convert the SSA form back to normal form, and
5. emit optimized iloc code

Note that the above is the minimum requirement. You are free to implement more optimizations if you would like.

## What to Turn In

- Submit your project through GitHub.
- Provide a `build` script that installs all necessary software and compiles and links (if necessary) your optimizer when run.
- Input files are in `input/*.il`.
- Create a bash script named `run.sh` that invokes your optimizer. If the name of the file is `input/prefix.il`, then your optimizer must write the
 optimized `iloc` to `output/prefix.dbre.il`. `run.sh` must run the optimized iloc file using `iloc -s output/prefix.dbre.il > output/prefix.out`. Finally, `run.sh` must run the following `diff` command with the file names parameterized based upon the input to `run.sh`

 ```bash
 diff -u --strip-trailing-cr input/prefix.out output/prefix.out || {
  echo "Output differs from expected output. See diff above."
  exit 1
}
```
